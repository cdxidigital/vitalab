# VitaLab Mobile Preview

A simplified Expo React Native preview of VitaLab with a score ring and quick activity logging.

## Run locally
1. Install Expo CLI:
   npm install -g expo-cli
2. Install deps:
   npm install
3. Start:
   npm start
4. Open in Expo Go (scan the QR) or run on web.
